# pgm-map-studio

A tool for creating and editing [PGM](https://github.com/PGMDev/PGM) Capture the Wool maps.

## Workflows

**Existing-map workflow** — import a map from a local folder, ZIP, or download URL, inspect and edit its regions, and export updated XML.

**Concept-first workflow** — draw a map layout from scratch using primitive shapes, place spawns and wools, and export a PGM XML file and basic world package.

## Installation

```bash
pip install -e .
```

## Usage

```bash
pgm-map-studio viewer        # open the map editor in your browser
pgm-map-studio import <path> # import a map from a folder or ZIP
pgm-map-studio --help        # list all commands
```
